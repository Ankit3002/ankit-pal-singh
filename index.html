Currently Doing
Open Source Contributor in CUPS

PR-> https://github.com/OpenPrinting/cups/pull/640

Fixed a bug where using the cupsctl command (a command for controlling the entire CUPS service) with command-line arguments would cause comments in the cupsd.conf file to disappear.

Created a private API in the CUPS codebase to fetch configurations along with comments, and I have implemented the functionality to set those configurations in the cupsd.conf file. Additionally, I have written a local API to handle the proper handling of escape '#' symbols while reading from the file.
